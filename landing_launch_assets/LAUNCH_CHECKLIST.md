# Landing Launch Checklist

## Copy

- [ ] Headline explains the problem in under 10 seconds.
- [ ] No official n8n affiliation implied.
- [ ] No guaranteed savings claim.
- [ ] No “security/compliance audit” claim.
- [ ] Caveats are visible.
- [ ] “Local/offline” is clear but not over-repeated.

## Design

- [ ] Desktop layout checked.
- [ ] Mobile layout checked.
- [ ] CTA visible above the fold.
- [ ] Sample report is readable.
- [ ] No generic AI visuals.
- [ ] No excessive cards/icons/template feel.

## Form

- [ ] Submit behavior tested.
- [ ] No secrets requested.
- [ ] Workflow JSON is not requested in the form.
- [ ] Confirmation message works.
- [ ] Endpoint configured or demo mode clearly shown.
- [ ] Safety note appears near the form.

## Privacy

- [ ] Local/offline explained.
- [ ] No credential inspection stated.
- [ ] Anonymization mentioned.
- [ ] No workflow upload unless intentionally built.
- [ ] Not affiliated with n8n stated.

## Technical

- [ ] Localhost works.
- [ ] Build works, if applicable.
- [ ] No console errors.
- [ ] Links work.
- [ ] Metadata present.
- [ ] Sample report page/section works.
- [ ] Privacy page/section works.

## Deploy

- [ ] Hosting target chosen.
- [ ] Domain or subdomain configured.
- [ ] Form endpoint configured.
- [ ] Canonical URL updated.
- [ ] Final smoke test completed.
- [ ] Test request submitted and received.
